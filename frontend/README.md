# CodeSense AI — Frontend

React + Vite client for CodeSense AI.

## Stack
- React 19, Vite
- Tailwind CSS v4 (via `@tailwindcss/vite`)
- React Router
- Zustand (state)
- Axios (HTTP client)
- Monaco Editor (`@monaco-editor/react`)

## Getting started

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173` by default. The backend is expected on `http://localhost:8080`.
